Elementary Cellular Automata

  • Start - Generate the pattern according to the given rule.
  • Rule (0 - 255) - Which rule to follow.
  • Cell size (1 - 10) - The size of each cell in pixels. Resets the canvas.
  • Initial cell ratio (0 - 100) - How many alive cells to generate when clicking the randomise button. A value of 0 will generate a single alive cell in the middle.
  • Randomise - Randomise initial generation of cells.
  • Reset - Reset to default configuration.
  • Theme - Choose between high and low contrast themes.

The Wikipedia page for Elementary Cellular Automata has pictures of the first 100 rules, try generating them for yourself.
